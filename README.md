A simple vampire villager game without an interface or view

This game was made using Java.

There are 8 players: 2 are vampires, and the others are villagers. When it becomes evening, vampires kill someone. The next day, they vote for someone they think is a vampire.
The bots choose a random person. The player chooses someone they believe to be a vampire. At the end of voting, the person with the most votes is eliminated.
If all villagers are dead, then vampires win. If vampires voted by villagers, then villagers win.

Play:
You can follow the instructions in the game! Have fun.

Note: If you are a villager, then the game is boring.
